# Models Directory

This directory stores trained machine learning models.

## Files

- `fraud_model.pkl` - Trained LightGBM fraud detection model (generated by train_model.py)

## Training

To train a new model:

```bash
python train_model.py
```

This will generate a new `fraud_model.pkl` file with the trained model.

## Model Details

- Algorithm: LightGBM Classifier
- Features: 9 engineered features
- Training: Class-weighted for imbalanced data
- Performance: <200ms inference time
